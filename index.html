Lab Guide Extension

Simulated Copilot Chat for hands-on training labs

What does this extension do?

The Lab Guide extension opens a simulated GitHub Copilot Chat panel inside VS Code with step-by-step instructions for each training lab. It does not replace or intercept real Copilot — it creates a separate webview panel that walks you through exercises.

Features

  • Opens a simulated chat panel with pre-scripted conversation steps
  • Provides Next → / ← Prev navigation between lab steps
  • Highlights relevant UI elements during each step
  • Launches course slides in the VS Code Simple Browser
  • Keyboard shortcuts: Ctrl+Shift+Right / Ctrl+Shift+Left

Commands

  • Lab Guide: Start Lab — pick a course and begin
  • Lab Guide: Next Step / Lab Guide: Previous Step
  • Lab Guide: Reset Lab — restart from step 1
✓ Transparent

What it does NOT do

We want you to feel confident installing this extension. Here's exactly what it accesses — and what it doesn't:

✗Does not access your files, folders, or workspace content ✗Does not send telemetry or analytics ✗Does not modify any VS Code settings ✗Does not intercept or proxy GitHub Copilot traffic ✗Does not access credentials, tokens, or secrets ✓Fetches lab.json from the course server to load step data ✓Opens a read-only webview panel (the simulated chat) ✓Opens course slides in VS Code's built-in Simple Browser

Source Code

The entire extension lives in the lab-guide/ folder of this repository. There are only three TypeScript source files:

lab-guide/
├── src/extension.ts — activation & command registration
├── src/chatPanel.ts — webview panel (simulated chat UI)
├── src/labController.ts — lab loading, step navigation
├── media/
│ ├── chat.css — chat panel styling
│ ├── chat.js — client-side message handling
│ └── copilot-icon.svg
└── package.json — extension manifest

No build artifacts are hidden. What you see is what runs.
